# 🎯 如何获取真实的"最后展示时间" - 给豆豆用

## ⚠️ 重要发现

**日期范围筛选器显示的日期不准确！**

所有网站都显示：
- 开始日期：2018年5月31日
- 结束日期：2025年12月18日（今天）

**这只是系统的默认查询范围，不是真实的广告投放日期！**

## ✅ 正确的获取方法

### 步骤1：搜索域名
访问：`https://adstransparency.google.com/?region=anywhere&domain={域名}`

例如：
- geckocustom.com: https://adstransparency.google.com/?region=anywhere&domain=geckocustom.com
- multicustomize.com: https://adstransparency.google.com/?region=anywhere&domain=multicustomize.com

### 步骤2：查看广告数量
页面会显示广告总数，例如：
- `~2000 个广告` - 大规模投放
- `3 个广告` - 小规模测试

### 步骤3：点击第一个广告（最重要！）
点击列表中的任意一个广告，进入"广告详情"页面

### 步骤4：查看"最后展示时间"
在广告详情页面，会显示：

```
最后展示时间：2025年3月30日
格式：文字广告
```

**这个日期才是真实的！**

### 步骤5：判断是否还在投放

计算"最后展示时间"到今天的天数：

```python
from datetime import datetime

last_seen = "2025-03-30"
today = datetime.now()
last_seen_date = datetime.strptime(last_seen, '%Y-%m-%d')
days_since_last_ad = (today - last_seen_date).days

print(f"停止投放：{days_since_last_ad}天")
```

### 步骤6：应用豆豆的客户分类

```
✅ 还在投放（0-7天前）
   - 如果开始时间 ≤ 30天 → 🔥 超新客户（TARGET）
   - 如果开始时间 31-180天 → ❄️ 冷却期（AVOID）
   - 如果开始时间 > 180天 → ♻️ 再营销（REMARKETING）

❌ 已停止投放（> 7天前）
   - 如果停止 180天+ → 🎯 可以重新激活的客户
   - 如果停止 < 180天 → ⏸️ 暂时不适合
```

## 🚀 批量检查流程

### 对于豆豆要批量检查店铺：

1. **准备域名列表**（从数据库导出）
2. **逐个访问 Google Ads Transparency**
3. **记录三个关键数据**：
   - 广告数量
   - 最后展示时间（点击第一个广告查看）
   - 广告主名称
4. **输入到识别系统**
5. **生成优先级列表**

### 建议的检查优先级：

1. **先检查自己数据库中的高价值店铺**
   - 月访问量 > 10,000
   - 有邮箱联系方式
   - 活跃状态（店铺还在运营）

2. **重点关注两类**：
   - 🔥 超新客户（0-30天内开始投放）
   - 🎯 已停止180天+的老客户（可重新激活）

## ⚡ 快速判断技巧

不需要点开每个广告，只需要：

1. **看广告数量**：
   - < 10个 → 小规模测试
   - 10-100个 → 中等规模
   - 100+个 → 认真在做
   - 1000+个 → 大客户

2. **看广告主数量**：
   - 1个 → 自运营（直客潜力⭐⭐⭐）
   - 2+个 → 有代理商

3. **点击第一个广告看日期**：
   - 最近7天 → 还在投放
   - 8天-180天 → 刚停止
   - 180天+ → 可重新激活
